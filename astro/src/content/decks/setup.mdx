This is the deck of the setup cards that will go to the various players.

<Aside title="Print and Play">
You may print this page to print the cards. It will print more accurately if you are using
Chrome (or Chrome-derived) browser. If you change the orientation to landscape and set the
margins to minimum (or custom), you should be able to fit 9 mini-cards on a page.

The cards will print with front and back aligned so they can be folded over and the corners
will be removed.
</Aside>
