:::danger[An important note]
_{frontmatter._["game.title"]}_ is currently a work in progress. While several of the design elements
are solid, others are still quite immature. The balance of the setup cards and the events, in particular,
is still being worked out. Feedback is appreciated; please leave it on the Wehrlegig Discord or the
[GitHub Discussions](https://github.com/brian-montgomery/jc2e-tea/discussions) associated with this project.
:::

This is currently __version {process.env.npm_package_version}__ of this design. For this version,
the [events](/jc2e-tea/components/decks/events/) and [smuggler cards](/jc2e-tea/components/decks/smuggler/)
have been reworked and the [setup cards](/jc2e-tea/components/decks/setup/) have been introduced.
Feedback on how those play, as well as any balance issues, is appreciated.

## Main Design Issues

In version 0.5.x, we're focusing on the following:

<div class="bold-list">
1. [__Setup Cards__](/jc2e-tea/components/decks/setup/).
   If any of the setup cards feels significantly under- or overpowered, please let us know.
2. [__Smuggler Cards__](/jc2e-tea/components/decks/smuggler/).
   Currently, the orders on the smuggler cards are relatively evenly distributed, but somewhat random. We're trying to determine how that feels versus making more "trade pathing" like the players are required to do.
3. [__Event Cards__](/jc2e-tea/components/decks/events/).
   -  _{frontmatter._["event.movement"]}_.
      We're trying to see if having the {frontmatter._["event.movement"]} at the beginning of some
      events (instead of at the end like the Elephant's March) helps make things simpler to run and
      doesn't require a piece. It lowers the possibility space of the events while also lowering
      the tension.
   -  _Current Balance._
      The balance of the events is still in flux. The amount of military action events have been increased and feedback on how that feels is appreciated.
4. [__Scenario End__](/jc2e-tea/rules/scenario-end/).
   There scenario currently ends when there are no colonized regions in America. While this feels
   like a logical place to end, the story feels somewhat abrupt. There are currently no special
   endgame scoring adjustments that happen (outside of the point penalties for region loss).
</div>
