A quick [Tabletop Simulator](https://www.tabletopsimulator.com/) Mod is available based on the original
[John Company 2e TTS mod](https://steamcommunity.com/sharedfiles/filedetails/?id=2474733631),
by Cole Wehrle, the designer of the original board game. This includes all of the current cards of
version __0.5.2__.

To install the mod, download the JSON file and [follow the instructions from the Tabletop Simulator
documentation](https://kb.tabletopsimulator.com/custom-content/importing-mods/#importing-saves).
