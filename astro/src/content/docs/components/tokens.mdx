### Speculation Tokens

Speculation tokens are used when the _Land Speculation_ Act gets passed. These function similarly
to loot tokens, although the money is split between the shares for the Land Company.

:::caution[Substitute]
Because these tokens aren't ready, we recommend using 8- or 10-sided die as a substitute for each of these.
:::

:::

## Markers

### Pawns for Tracks

You will need a couple of markers (or pawns) for the various tracks on the
[scenario board](/jc2e-tea/components/board/). These can be anything that are available.

### Region Strength

Region strength for foreign-settled colonies are represented using 6-sided dice of various
colors. The colors will show which regions are allied (friendly). __A region's strength will
never go below one.__
